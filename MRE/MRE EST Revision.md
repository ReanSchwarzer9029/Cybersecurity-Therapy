## Disassemblers
- Disassembles translate binary machine code to readable assembly
- Reversing tools:
	- IDA Pro
	- OllyDbg
	- CFF Explorer
	- RASM2
	- WinDbg
- Disassemblers attempt to find and understand Opcodes and Operands from a list of binary values to hexadecimals
